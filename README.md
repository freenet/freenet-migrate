# freenet-migrate

Reusable contract & delegate **upgrade-migration** for [Freenet](https://freenet.org)
dApps: state and secret carry-forward across WASM version changes.

The `freenet-migrate` and `freenet-migrate-build` crates are being imported in
the first pull request. Horizon-A step A3 of the graceful-upgrades design,
[freenet-core#2776](https://github.com/freenet/freenet-core/issues/2776).
